Projeto Verificador de idade

Projeto construido para informar a idade do usuario usando JavaScript.

Saiba mais...

Este projeto foi desenvolvido para resolver a necessidade de calcular e exibir a idade de um usuário de forma dinâmica e interativa. O raciocínio começou com a identificação de um problema comum em formulários web: a falta de validação e feedback imediato sobre dados pessoais. Para isso, utilizei JavaScript para capturar a data de nascimento inserida pelo usuário, calcular a idade em tempo real e exibir o resultado de maneira clara. Tecnologias empregadas incluem HTML para a estrutura, CSS para o estilo e JavaScript para a lógica de cálculo, garantindo uma experiência responsiva. O desafio principal foi lidar com validações de datas inválidas, o que foi superado com condicionais e tratamento de erros. Esse projeto reforçou conceitos fundamentais de manipulação de DOM e interatividade, servindo como base para aplicações mais complexas.

Projeto Android

Projeto construido para treinar conteudos e imagens dinâmicas.

Saiba mais...

O objetivo deste projeto foi criar uma página web simulando uma interface de aplicativo Android, focando no treinamento de conteúdos dinâmicos e manipulação de imagens. O raciocínio partiu da necessidade de praticar técnicas de design responsivo e interatividade, comuns em apps móveis. Utilizei HTML para estruturar os elementos, CSS para estilizar a aparência (inspirada no Android) e JavaScript para adicionar dinamismo, como troca de imagens e animações simples. O desafio foi garantir que as imagens fossem carregadas eficientemente e que a interface se adaptasse a diferentes dispositivos, o que foi resolvido com media queries e otimização de assets. Esse exercício aprimorou habilidades em front-end, especialmente em criar experiências imersivas e preparou o terreno para desenvolvimento de apps reais.

Projeto divulgação de redes sociais

Projeto construido para divulgar links de uma empresa.

Saiba mais...

Este projeto foi concebido para atender à demanda de empresas por uma presença online simples e eficaz, focando na divulgação de links de redes sociais. O raciocínio iniciou com a análise de como pequenas empresas podem expandir seu alcance digital sem plataformas complexas. Desenvolvi uma página limpa e acessível usando HTML para a estrutura básica, CSS para um design atrativo e responsivo, e JavaScript para adicionar interatividade, como botões de compartilhamento. O principal desafio foi otimizar para dispositivos móveis, assegurando que os links fossem fáceis de acessar e compartilhar. Tecnologias como Font Awesome foram integradas para ícones visuais. O resultado é uma ferramenta prática que facilita a conexão entre empresas e clientes, reforçando conceitos de UX/UI e marketing digital.

Projeto Login

Projeto construido para treinar o uso de Html5 e Css3 para criação de uma tela de login.

Saiba mais...

O foco deste projeto foi praticar a criação de uma tela de login funcional e esteticamente agradável, utilizando as últimas versões do HTML e CSS. O raciocínio começou com a identificação da importância de interfaces de autenticação seguras e intuitivas em aplicações web. Estruturei o layout com HTML5 para semântica aprimorada, aplicando CSS3 para estilos modernos, como gradientes, sombras e transições suaves, sem necessidade de JavaScript para funcionalidades básicas. O desafio foi garantir acessibilidade e responsividade, resolvido com técnicas como flexbox e validações nativas de formulários. Esse exercício consolidou conhecimentos em design front-end, preparando para integrações com back-end e enfatizando boas práticas de segurança e usabilidade.
